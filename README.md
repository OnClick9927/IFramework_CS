# IFramework_CS
Interface | Inject | Interesting | Increase  C#
``` csharp
while(true)
    Console.Write("Thanks For EveryOne Who Used It Once !")
```
## 整体内容
* 数据绑定
* 配置绑定
* 迭代器协程
* 具有优先级的消息转发
* 操作记录与撤销
* 可观察对象
* 可回收对象
* Packet（自定义数据协议）
* NodeAction（节点事件）
* Net（TCP，UDP，WS，Http，kcp）
* Serialization（string版本【主要用于csv】，二进制版本【支持简单的数据结构】）
* Singleton（通用单例）
 * Utility（工具）
